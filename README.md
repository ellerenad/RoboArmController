# RoboArmController

Repository for the Roboarm controller

## Prerequisites:

- gazebo http://gazebosim.org
- leap motion sdk (for the hands sensor)
- maven



##Start leapmotion

sudo leapd

###Other important commands

cd .Leap\ Motion/
Visualizer
LeapControlPanel
LeapControlPanel --showsettings
